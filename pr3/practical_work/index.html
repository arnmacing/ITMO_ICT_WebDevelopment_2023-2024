    1. Найдите всех водителей с именем "Олег" (или любым другим именем на ваше усмотрение).

    1. Возьмите любого случайного владельца, получите его id, а затем получите экземпляр удостоверения в виде объекта модели по этому id (можно в 2 запроса).

    1. Выведите всех владельцев красных машин (или любого другого цвета, который у вас присутствует).
